# Decision Tree on Iris Dataset

This project applies a **Decision Tree classifier** to the classic Iris dataset using Python and Scikit-learn.

## Dataset

- **Source:** Scikit-learn built-in Iris dataset
- **Features:**  
  - sepal length (cm)  
  - sepal width (cm)  
  - petal length (cm)  
  - petal width (cm)
- **Target Classes:**  
  - setosa  
  - versicolor  
  - virginica
- **Samples:** 150 instances

## Accuracy : 96%
